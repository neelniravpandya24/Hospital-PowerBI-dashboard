# 🏥 Hospital Management Dashboard

## 📌 Project Overview

The Hospital Management Dashboard is an interactive Power BI project developed to analyze hospital operations, patient records, doctor performance, financial statistics, and resource utilization. The dashboard transforms raw hospital data into meaningful insights using interactive visualizations, KPI cards, and dynamic reports to support better decision-making.

---

## 🎯 Objectives

- Monitor hospital performance using key metrics.
- Analyze patient demographics and treatment records.
- Evaluate doctor and department performance.
- Track hospital revenue and financial trends.
- Optimize hospital operations and resource management.

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI Desktop
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
